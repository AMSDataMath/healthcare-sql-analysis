# Healthcare SQL Analysis Portfolio

## Project Overview

This project demonstrates my SQL skills using a realistic healthcare dataset containing approximately 55,000 patient records.

The purpose of this project is to showcase my ability to clean data, explore datasets, answer business questions, and perform healthcare analytics using PostgreSQL.

---

## Dataset

Table Used:

- healthcaredata2

Additional Lookup Tables:

- doctors

Database:
