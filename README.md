# Healthcare SQL Analysis Portfolio

## Project Overview

This repository showcases SQL analysis using a realistic healthcare dataset containing approximately 55,000 patient records. The project demonstrates how SQL can be used to answer business questions that healthcare organizations face every day.

## Business Questions Answered

### Hospital Analysis
- Which hospitals admitted the most patients?
- Which hospitals generated the highest total billing revenue?
- Which hospitals have the highest average billing amount?
- Which hospitals handled the most emergency admissions?
- Which hospitals treated the highest number of diabetic patients?
- Which hospitals have the oldest average patient population?

### Doctor Analysis
- Which doctors treated the most patients?
- Which doctors generated the highest total billing?
- Which doctors have the highest average billing amount?
- Which doctors handled the most emergency admissions?
- Which doctors treated the most diabetic patients?
- What is the average patient age for each doctor?

### Insurance Analysis
- Which insurance providers cover the most patients?
- Which insurance providers generated the highest total billing?
- Which insurance providers have the highest average billing amount?
- Which insurance providers handled the most emergency admissions?
- Which insurance providers cover the most diabetic patients?

### Patient Analysis
- What is the age distribution of patients?
- Which blood types are most common?
- Which medical conditions occur most frequently?
- Which medications are prescribed most often?
- What are the most common admission types?

### Billing Analysis
- What is the total billing amount?
- What is the average billing amount?
- Which medical conditions have the highest treatment costs?
- Which hospitals generate the highest revenue?
- Which doctors generate the highest revenue?

## Purpose

The goal of this project is to demonstrate practical SQL skills for healthcare data analytics by solving realistic business problems using PostgreSQL. The analyses in this repository simulate reporting tasks commonly performed by healthcare data analysts and support data-driven decision-making.
